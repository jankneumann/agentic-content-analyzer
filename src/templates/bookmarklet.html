Save to Newsletter Aggregator

Install this bookmarklet to save any webpage with one click.

Install

1 Drag to your bookmarks bar

Drag the button below to your browser's bookmarks bar:

Save to Aggregator

Click and drag the button above into your bookmarks bar.

2 Use it on any page

Visit any webpage, optionally select some text, then click the bookmarklet in your bookmarks bar. A save form will open with the URL, title, and selected text pre-filled.

Mobile browsers: On iOS Safari, create a regular bookmark first, then edit it and replace the URL with the bookmarklet code below.

How It Works

The bookmarklet captures the current page URL, title, and any selected text, then opens the save form. The content is queued for extraction and processing through your Newsletter Aggregator pipeline.
