iOS Shortcut

Save any URL from the iOS Share Sheet to your Newsletter Aggregator.

What This Does

This shortcut adds a "Save to Aggregator" option to your iOS Share Sheet. When you find an interesting article in Safari, a news app, or anywhere else, tap Share and select the shortcut to save the URL for processing through your Newsletter Aggregator pipeline.

Setup Steps

1 Open the Shortcuts app

Open the Shortcuts app on your iPhone or iPad. Tap the + button in the top-right corner to create a new shortcut.

2 Accept Share Sheet input

Tap Add Action, search for "Receive" and select Receive input from Share Sheet. Set the input type to URLs.

3 Add the API request

Add a Get Contents of URL action. Configure it as follows:

URL:

Method: POST

Headers:

Content-Type: application/json

Request Body (JSON):

{ "url": [Shortcut Input], "source": "ios_shortcut" }

For the "url" value, use the Shortcut Input variable (tap the field and select it from the variable list).

4 Show confirmation

Add a Show Result action and set it to display the output from the previous step. This will show a confirmation when the URL is saved successfully.

5 Name and enable in Share Sheet

Tap the shortcut name at the top and rename it to "Save to Aggregator". Tap the settings icon and enable Show in Share Sheet.

Configuration

Adding API Key Authentication:

If your server requires authentication, add an extra header to the Get Contents of URL action in Step 3:

X-Admin-Key: your-api-key-here

Replace your-api-key-here with the value of your ADMIN_API_KEY setting.
